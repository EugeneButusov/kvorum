API · v1.0

REST + JSON. Authorization: Bearer kv_live_…. All responses are tabular-numeric, paginated by cursor, and timestamped at the source block. SDKs in typescript, python, rust.

GET /proposals?dao={slug} ✓ stable

List proposals across DAOs. Filter by dao, state, since. Returns AI summary inline if summary=full. Bandwidth: ~2.4 KB/proposal.

Query parameters

ParameterTypeDescription
dao*stringDAO slug, e.g. compound, uniswap. See /daos for full list. Use * for cross-DAO.
stateenumOne of active, passed, executed, defeated, queued, draft.
sinceISO-8601Inclusive lower bound on created_at. Default: 90 days ago.
summaryenumnone (default) · tldr · full. Each level adds latency.
limitint1–100. Default 20.
cursorstringFrom previous response's next_cursor. Opaque.
