Get proposal

GET /v1/proposals/{dao_slug}/{proposal_id}

Returns a single proposal with its full description, on-chain calldata, action breakdown, latest tally, mismatch analysis, and provenance metadata. The AI-summary block is included when available; absence indicates either the summary has not been computed or confidence dropped below the publish threshold (§6.18).

Auth. Public endpoint. An API key is required only above the anonymous rate limit (60 req/min). Pass it as Authorization: Bearer kvm_live_….

Path parameters

NameTypeDescription
dao_slugrequiredstringSlug of the DAO. One of compound, aave, lido.
proposal_idrequiredstringThe DAO-native proposal identifier (numeric for Compound/Aave; uuid for Lido Easy Track).

Query parameters

NameTypeDescription
includestring[]Comma-separated. One or more of tally,actions,mismatches,summary,forum. Default: all.
formatenumjson (default) or json+sse for live subscription via SSE channel.
